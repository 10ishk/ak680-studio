# Security Policy

## Reporting Issues

Please report security or hardware-safety issues through the repository issue tracker or by contacting the maintainer privately if public disclosure could put users at risk.

## Hardware Safety

Hardware-write functionality can damage user devices if implemented incorrectly. Any future code that writes to the keyboard must receive careful review, include a Red Team test plan, and require explicit maintainer approval before it is merged.

Work Package 1 contains read-only local JSON inspection only.
